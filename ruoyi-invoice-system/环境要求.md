# RuoYi 发票管理系统 - 环境要求

## 系统要求

### 操作系统
- Windows 10/11, Windows Server 2016+
- Linux (Ubuntu 18.04+, CentOS 7+)
- macOS 10.14+

### 硬件要求
- 内存: 最低 2GB，推荐 4GB+
- 磁盘空间: 最低 1GB
- CPU: 双核处理器

## 软件依赖

### Java 环境
- JDK 1.8 或更高版本
- 推荐使用 OpenJDK 8 或 Oracle JDK 8

### 数据库
- MySQL 8.0 或更高版本
- 或 MariaDB 10.3+

### 构建工具
- Maven 3.6 或更高版本

### 前端开发环境 (可选)
- Node.js 16+
- npm 或 yarn

## 环境变量配置

### Windows 环境变量
```batch
set JAVA_HOME=C:\Program Files\Java\jdk1.8.0_xxx
set MAVEN_HOME=C:\apache-maven-3.8.x
set PATH=%JAVA_HOME%\bin;%MAVEN_HOME%\bin;%PATH%
```

### Linux/macOS 环境变量
```bash
export JAVA_HOME=/usr/lib/jvm/java-8-openjdk
export MAVEN_HOME=/opt/apache-maven-3.8.x
export PATH=$JAVA_HOME/bin:$MAVEN_HOME/bin:$PATH
```

## 端口要求

- 后端服务端口: 8080 (可配置)
- 数据库端口: 3306
- Redis端口: 6379 (可选)

## 网络要求

- 可访问 MySQL 数据库
- 可访问 Maven 中央仓库
- 文件上传下载需要足够的网络带宽

## 安装验证

### 验证 Java 安装
```bash
java -version
javac -version
```

### 验证 Maven 安装
```bash
mvn -version
```

### 验证 MySQL 安装
```bash
mysql --version
```

## 可选组件

### Redis (用于缓存和会话管理)
- Redis 5.0+
- 可选安装，提升系统性能

### Nginx (用于反向代理和静态资源)
- Nginx 1.18+
- 可选安装，用于生产环境部署
